九月的最后一天

发表于 2018-09-30 | 分类于 随笔

我觉得我挺后悔浪费了好多的时间，以致于好长一段时间都是向睡眠要时间，然后同时又怕死，想按时休息。应该认识的问题是，时间永远都不会是够得，能做的就是提高时间的利用率和做事的效率。然后状态就是很累，却可以刷视频很久，哪怕下一秒就要睡着也要开着手机，放空自己。不去思考。然后带着懊悔，牺牲睡眠，做些些微的补救。再后来就是没有补救，无法感受到内疚的心情，只觉得这样是不对的。总之就是很累，对事情很麻木。

第一件当是生存的压力，如若自己能力不足，特别与别人对比，别人威胁到自己。

第二件是关于自己的规划提升和兴趣之间，我多了一种状态。比提升多了一点兴趣，比兴趣少了一点有趣。

第三件是过去的调节使自己不会懊悔，而是抓住当下。然后只抓住了前半段，后半段不见了踪迹。

第四件是在环境相处的不自在。

最后就是，事情是可以调节的，大概我的中枢结构，缺乏明确的目的和相应的自制力，脱离了工作环境，在住的环境里，就会不敢不顾，放空大脑，看些没营养的视频。

应该保持的心态就是，不喜不悲。

同时明确下自己目标以及动机：

一条能够贯穿我现在及未来工作乃至人生主线的兴趣爱好，这将有效的统合我的时间和生命，也是我能继续下去的动力源泉。

为此，我将需要提供物质基础而努力工作，并将这条主线融入我生命中的每一部分。

时间依然宝贵，就如生命，不只是为这条主线赋予的时间跨度只有这短短一生而不能穷尽，也是身处社会关系中，以社会的一般规则要求的紧迫。

这样，我的情绪将不再是失落的，失控的，不能管理的，因为我的大脑已经有了需要他关注的事情。

关于情绪

发表于 2018-09-28 | 分类于 随笔

看着别人能力不如自己，工资却比我高，一般来说，任何人都会感到不公平，不满。我自己体会具体情况，同一个公司的那就直接走了。如果是不同地方，大城市比小城市工资高，要看到客观事实。

我近来遇到的生活中的负面情绪，是很大的一部分，影响我许多方面了。是合租房的问题，平摊的水电费，别人偷偷住了几个人，我却要为别人买单，关键是对方素质差，消息时间各种影响我休息，不关灯等等。这个情绪我憋了两个多月了，现在总算谋划着搬走了。

在往前就是工作的压力，这段情绪是长期，近两年的时间，现在想来竟然一直没调节，还是处在不断发作的循环中也是很神奇。一是业务能力的不熟悉，对手上的项目无从获取帮助，智能不停思索百度解决了，期初压力很大。后期就是业务上项目，一个个规划不好，导致结果差强人意，到我程序开发阶段，实在是影响自己的自信心。再其次，在这其中与别人的相处问题。

细节上，在与别人相处却是很敏感的状态，别人不会我消息，我会不让停地去看有没有回，会失落，会对别人产生依赖。与陌生人相处仍然会紧张，仍然会在意别人的看法。

这就是我的情绪。

什么是人工智能

发表于 2018-09-17 | 分类于 人工智能

李开复《人工智能》笔记

人工智能就是能够对环境做出响应，并采取行动以达到目标的主体的研究探索。

不能要求人工智能要和人类一样，因为人类的大脑运作方式尚未研究明白，而且当前的机器学习是通过的大量的数据进行训练。比方说图片识别，计算机可能通过大量的自行车图片去认识自行车，而人类可能只需要看过一次就可以识别所有的自行车。计算机没有人类的思维的抽象能力，以人类的能力来要求计算机注定是失败的，对于整个人工智能的发展也是有害的。

应当将人工智能看做独立于人类之外的一个存在，尽管在许多方面有着相似之处。

关于机器学习、深度学习

发表于 2018-09-12 | 分类于 人工智能

首先，机器学习属于人工智能分支，深度学习属于机器学习分支。

相比我们写代码，无论前端，后端，程序都是死的，只会按固定的逻辑去走，不会有什么变化。

而机器学习，相比人类的学习，通过大量的数据练习，或监督学习，或无监督学习，或强化学习，实现诸如在一张图片中识别对象，无人驾驶，智能语音等传统编程做不了的应用。

深度学习则是将神经网络引入机器学习产生的新的分支。

关于人工智能

发表于 2018-09-08 | 分类于 人工智能

以下是对所学习了解的知识做的一个总结。

毫无疑问，近来云计算，大数据，人工智能很火。但是这些东西，并不是突然就冒出来的。

人工智能早在上个世纪六十年代就已经提出来，而云计算和大数据的概念，在十年前。

得益于技术进步，CPU，存储，网络的快速发展，基于虚拟化和调度的云计算便有了发展的舞台，网络的发展带来的海量数据以及数据
存储，如何利用这些数据催生了大数据。而人工智能，在前两者的基础上，依托大量数据，训练机器去学习并记忆人的处理方式。

人工智能最先运用甚至颠覆的行业：

互联网、金融、教育（李开复）

最有可能被淘汰的岗位：

目前机器是不擅长的情感，不具备创新等人类特有的性质。所以，从事文学创作等精神劳动的不易被淘汰？从新闻里了解，无人驾驶
已经有应用了，未来是不是司机这个岗位就要被替代了。就像功能机到智能机，短短几年就已经翻天覆地，不存在平滑过渡。因为
优势太明显了。

十年？二十年？不敢想象。

人工智能学习路线：

1，数学相关 -微积分、线性代数、离散数学、概率与数理统计

2，python相关

3，机器学习

4，深度学习

关于应试的举措

发表于 2018-08-22 | 分类于 随笔

每次截止日期，都异常紧张，每个人都是如此。在稍微宽松，看不到未来的急迫的时候，想着是不是应该放弃，等下次
准备得充分之后再进行尝试。

这样的想法尤其不可取，寄希望于下一次，只会使得下次的压力增大。其次，哪有那么多的下次可以让自己挥霍。

如此，就需要准备好这一次。怎么办？时间不够，精力不够？如何在有限的时间精力下，掌握更多的知识。

事实上单是通过考试，并不需要那么多的积累。以考试而论，只需要通过开始的那部分即可。尽管考试不是目的，但
开始一道闪不过去的门槛。连门槛都过不去，又如何谈其他。

从此以后的行动应该是这样的，针对考试题型，训练自己的解题能力，其次根据考试的要点，有针对性的去分篇章复习相关
的知识点。

再其次，在此基础上，剩下的时候，就是踏实的从基础开始学起，弥补和那些系统学习的人之间的差距。

所谓曲线救国。

论自己的桌面工作环境

发表于 2018-08-20 | 分类于 随笔

过去一段时间一直有点执拗，觉得linux大法好，新买的电脑因为驱动不兼容等各种原因，安装过archlinux,ubuntu,kalilinux,
centos等等，最后安装的linuxmint对自己的电脑兼容最好，即使是如此，对于桌面环境，我用的xfce,还是重装了好几遍。

使用linux，少不得需要一颗爱折腾的心。不过现实是没那么多时间，今天看了freeBSD吧上说的，觉得挺有道理。
论使用便宜程度，windows下的记事本要比VIM好太多。图形化的操作发展，就是为了提升人的工作效率。vim存在于那个
命令行年代，没有鼠标，有其存在背景。

没有必要带着LINUX优越感，看不起Windows下的盗版之类的。开源的含义意味着自由。事实上现在的linux的桌面环境还是
要比windows,MAX OX 差很多。LINUX的核心在于内核，也就是UNIX的翻版。

基于此，个人以后主设备用windows7,也可以打游戏，其次旧电脑用freebsd用于学习UNIX,服务器环境用LINUX.
